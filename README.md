<h2 id= "description">Description</h2>
A simple react application to display 25 random eployees, and non-sensitive information about them. A user can sort the employees, and search through them by name.
    
<h2 id= "technology">Technology Stack</h2>
 Node.Js, React, GIT, GitHub, APIs
  
<h2 id= "usage">Usage</h2>
To search and sort through a table of employees.
